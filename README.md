# WordCount-con-Hadoop-MapReduce-Actividad-Semana-10
Implementación de un job **MapReduce** clásico (conteo de palabras) sobre un conjunto de documentos de texto, desarrollado en dos variantes:  1. **Java** — API nativa de Hadoop (`org.apache.hadoop.mapreduce`). 2. **Python (Hadoop Streaming)** — `mapper.py` / `reducer.py` ejecutables vía `hadoop-streaming.jar`.
